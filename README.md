# Usage
<!-- usage -->
```sh-session
$ npm install -g todo-bash
$ todo COMMAND
running command...
$ todo (-v|--version|version)
todo-bash/0.0.0 darwin-x64 node-v8.11.4
$ todo --help [COMMAND]
USAGE
  $ todo COMMAND
...
```
